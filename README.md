# README
Railsのwebpackオプションを用いて作成したサンプルアプリです  
* Ruby version  
2.4.1  
* System dependencies  
rails: 5.1  
database: mysql  
* Database creation  
```rails db:create```  
* Database initialization  
```rails db:migrate db:seed```  
* How to set up  
```rails webpacker:install:react```  
* Services (job queues, cache servers, search engines, etc.)  
start server  
```bundle exec foreman start```
